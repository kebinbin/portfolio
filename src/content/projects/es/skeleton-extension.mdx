<TextBlock title="Detrás de este proyecto">Quería crear un paquete para inyectar colores automáticamente a los elementos HTML, en lugar de hacerlo manualmente. Es una función sencilla, pero muy útil para agilizar el proceso de diseño y desarrollo de diseños HTML/CSS, detectar fácilmente errores y desalineaciones, e incluso ver cómo se diseñan los elementos HTML en tus sitios web favoritos. Este plugin empezó como un paquete npm que inyectaba clases CSS en tu proyecto, pero resultó más práctico tenerlo como plugin en el navegador, evitando así tener que añadir dependencias a tus proyectos JavaScript.</TextBlock>
